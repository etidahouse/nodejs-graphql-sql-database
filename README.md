# nodejs - graphql - sql database

## Init

`npm init -y`  
`yarn add express express-graphql graphql`  
`yarn add @types/express @types/express-graphql @types/graphql nodemon -D`  
`npx tsconfig.json`
`sudo npm install typescript -g`

## Run

`yarn build`  
`yarn start`  

App deployed at http://localhost:8080
